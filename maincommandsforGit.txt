1. в терміналі в папці з проектом D:\workwithgit ініціалізуємо Git,
щоб він слідкував за змінами в проекті(створення нових файлів) за допомогою команди
git init
2.Перший раз потрібно встановити все глобально --global
Налаштування локальних конфігурацій
git config --local user.name "Victor"
git config --local user.email example@gmail.com

git status - статус 
щоб Git почав слідкувати за змінами в файлах проекту(наприклад у всіх index.html, style.css, script.js)
3. git add -A
Після команди git status він слідкує(за змініми) за зеленими файлами і бачить нові створені червоні 
файли за змінами якимх не слідкує
Для додавання нових файлів за якими буде слідкувати git можна використати git add -A або
gid add style1.css  або *.css
4. Додавання комітів
git commit -a -m 'message for my first commit'
5. Які були зроблені коміти
git log
- створюємо новий репозиторій на github.com
- git branch -M main
- git remote add origin https://github.com/victordrach1/18NumStrPractice.git
- git push -u origin main
6. git push  відправляєм на репозиторій
